# 🌱 EcoPower Quest

**EcoPower Quest** is a 2D educational platformer game developed in Unity. Designed specifically for the custom "+Power" console, the game integrates environmental education and basic mathematics across three distinct difficulty levels.

## 🎮 Features
* **Educational Theme:** Blends environmental awareness with math puzzles to create an engaging learning experience.
* **Progressive Difficulty:** Includes 3 distinct levels (Easy, Medium, Hard) tailored for continuous learning.
* **Minimalist Control Scheme:** Fully playable using strictly three keys, adhering to the +Power console hardware guidelines.

## 🕹️ Controls
The game mechanics are built around a strict 3-button input system:
* **`TAB`** : Move Left / Previous Item
* **`BACKSPACE`** : Move Right / Next Item
* **`ENTER`** : Jump / Confirm Selection

## 🛠️ Built With
* **Game Engine:** Unity 6
* **Language:** C#
* **Platforms:** Windows (PC) & WebGL
